# 🌸 Iris Flower Classification using Machine Learning

An end-to-end Machine Learning project that classifies Iris flower species into **Setosa**, **Versicolor**, and **Virginica** based on their sepal and petal measurements (length and width). 

---

## 📌 Project Overview
This project utilizes the classic **UCI Iris Dataset** to train a classification model. Using Python's robust data science ecosystem, the data is processed, split, and fed into a **Logistic Regression** algorithm, achieving a high predictive accuracy of **96.67%**.

---

## 🚀 Features
* **Data Exploration:** Seamless loading and exploration of data using Pandas DataFrames.
* **Efficient Splitting:** Stratified data splitting ($80\%$ training, $20\%$ testing) to preserve class balance.
* **High Performance:** Leverages Logistic Regression for fast and highly accurate multiclass classification.
* **Comprehensive Metrics:** Includes detailed evaluation reports featuring Precision, Recall, F1-Score, and Overall Accuracy.
* **Real-time Prediction:** Pre-configured pipeline to predict species for any custom user input data.

---

## 🛠️ Technologies Used
* **Language:** Python
* **Machine Learning:** Scikit-Learn
* **Data Manipulation:** NumPy, Pandas
