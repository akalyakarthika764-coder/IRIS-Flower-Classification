# IRIS-Flower-Classification
An Iris Flower Classification project uses Machine Learning (Logistic Regression) to predict flower species (Setosa, Versicolor, Virginica) based on sepal and petal length/width. Using Python, Scikit-Learn, and Pandas, the dataset is split into 80% train and 20% test data. The trained model achieves a high accuracy of 96.67%.
